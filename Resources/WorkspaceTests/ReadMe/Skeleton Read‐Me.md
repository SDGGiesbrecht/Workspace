# MyProject

## Importing

`MyProject` is intended for use with the [Swift Package Manager](https://swift.org/package-manager/).

Simply add `MyProject` as a dependency in `Package.swift`.

`MyProject` can then be imported in source files:

```swift
import MyProject
```
