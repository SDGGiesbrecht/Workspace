Importing

CheckedInDocumentation provides libraries for use with the Swift Package Manager.

Simply add CheckedInDocumentation as a dependency in Package.swift and specify which of the libraries to use:

let package = Package( name: "MyPackage", dependencies: [ .package( name: "CheckedInDocumentation", url: "http://example.com", from: Version(1, 0, 0) ), ], targets: [ .target( name: "MyTarget", dependencies: [ .product(name: "CheckedInDocumentation", package: "CheckedInDocumentation"), .product(name: "Extensions", package: "CheckedInDocumentation"), ] ) ] )

The modules can then be imported in source files:

import CheckedInDocumentation import Extensions
