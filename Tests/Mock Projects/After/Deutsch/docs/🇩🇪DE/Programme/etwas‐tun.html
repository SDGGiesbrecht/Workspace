Unterbefehle

hilfe

Zeigt Gebrauchsinformationen an.

sprache‐einstellen [Spracheinstellung]

Stellt die Spracheinstellung ein. (Das Argument weglassen, um an die Systemeinstellungen zurückzufallen.)

unterbefehl [Zeichenkette]

Ist ein Beispielsunterbefehl.

version

Zeigt die verwendete Version an.

zwischenspeicher‐leeren

Entfernt zwischengespeicherte Daten.

Optionen

•ohne‐farben

Blendet Farben aus der Ausgabe aus.

•sprache [Spracheinstellung]

Eine Sprache anstelle deren zu verwenden, die in den Einstellungen angegeben ist.

Argumentarte

[Spracheinstellung]

Eine Liste IETF Sprachbezeichnungen oder SDGCornerstone‐Sprachsymbole. Doppelpunkte geben die Ersatzreihenfolge an. Kommata geben an, dass mehrere Sprachen verwendet werden sollen. Beispiele: „de-DE“ oder „🇩🇪DE“ → Deutsch aus Deutschland, “bar,de;fr” → beide Bairisch und Deutsch, sonst Französisch
