## Εισαγωγή

`PartialReadMe` προορίζεται για χρήση με τον [διαχειριστή πακέτων του Σουιφτ](https://swift.org/package-manager/).

Πρόσθεσε τον `PartialReadMe` απλά στο κατάλογο των εξαρτήσεων στο `Package.swift`:

```swift
let πακέτο = Package(
    name: "ΠακέτοΜου",
    dependencies: [
        .package(url: "https://somewhere.com", .upToNextMinor(from: Version(0, 1, 0))),
    ],
    targets: [
        .target(name: "ΣτόχοςΜου", dependencies: [
            .productItem(name: "PartialReadMe", package: "PartialReadMe"),
        ])
    ]
)
```

Έπειτα `PartialReadMe` μπορεί να εισάγεται στα πηγαία αρχεία:

```swift
import PartialReadMe
```
