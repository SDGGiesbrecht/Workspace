Einführung

PartialReadMe stellt eine Bibliotek für zur Verwendung mit dem Swift‐Paketenverwalter (Swift Package Manager) bereit.

PartialReadMe einfach als Abhängigkeit (dependency) in Package.swift hinzufügen:

let paket = Package( name: "MeinePaket", dependencies: [ .package(url: "http://example.com", .upToNextMinor(from: Version(0, 1, 0))), ], targets: [ .target(name: "MeinZiel", dependencies: [ .productItem(name: "PartialReadMe", package: "PartialReadMe"), ]) ] )

Das Modul können dann in Quelldateien eingeführt (imported) werden:

import PartialReadMe
