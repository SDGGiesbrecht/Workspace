Importing

PartialReadMe provides a library for use with the Swift Package Manager.

Simply add PartialReadMe as a dependency in Package.swift:

let package = Package( name: "MyPackage", dependencies: [ .package(url: "http://example.com", .upToNextMinor(from: Version(0, 1, 0))), ], targets: [ .target(name: "MyTarget", dependencies: [ .productItem(name: "PartialReadMe", package: "PartialReadMe"), ]) ] )

The module can then be imported in source files:

import PartialReadMe
